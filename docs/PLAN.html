00-1 Firestore 데이터 구조
컬렉션 경로 저장 필드 설명
users/{uid}/books/{bookId} title, fileHash, totalChars, createdAt 소설 메타데이터
…/books/{bookId}/progress chapterIdx, scrollY, updatedAt 읽기 위치 (실시간 저장)
…/books/{bookId}/bookmarks chapterIdx, scrollY, label, createdAt 북마크 목록
…/books/{bookId}/highlights startOffset, endOffset, color, note 하이라이트 + 메모
users/{uid}/settings theme, fontSize, lineHeight, fontFamily 전역 독자 설정

* 소설 파일 자체는 서버에 저장하지 않음. 파일명 + 크기로 생성한 해시값(bookId)으로만 식별.
